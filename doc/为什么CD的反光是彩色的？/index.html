    CD反光会色散的原因要从其微结构说起.

    (图片来自http://en.wikipedia.org/wiki/Compact_Disc )

    如图, CD一般是4层结构, 从下倒上依次是:
    A. 聚碳酸酯 (无色透明) 做成的光盘基片. 基片的内表面有用于记录数据的螺旋形排列的凹槽 (见下图).
    B.涂在基片内表面之上的金属反光涂层 (一般为铝).
    C. 保护涂层.
    D. 光盘印刷封面.


    (图片来自http://en.wikipedia.org/wiki/Compact_Disc )

    扫描电子显微镜下的微结构图像:

    (图片来自http://www.myce.com/review/pioneer-bdr-205-blu-ray-drive-review-22946/DVD-Blu-Ray-Reading-Performance-9/ )


    而光盘的螺旋形排列的凹槽构成了近似周期结构的反射型衍射光栅. 光盘反光的色散正是来源于光栅衍射.
    所谓光栅就是具有周期性结构的衍射屏. 如下图, 光垂直入射到一个反射光栅上. 每个衍射射点都相当于一个次波源, 空间中的光场分布就是所有次波源相干叠加的结果. 由于光栅具有周期性 (栅格周期间隔为 ), 所以相邻周期栅格间相同方向 的光具有固定的光程差: . 若光程差正好为波长的整数倍, 则所有周期在 的光相干增强,

    也就是说对于 波长 的光, 第 n 级衍射峰发生在

    的角度. 其中第 0 级衍射 对应几个光学平面反射的位置.

    光盘的凹槽位于间距 (比可见光波长略大, 因此CD对可见光会有比较明显的衍射现象) 的环形轨道上, 这些环形轨道就形成了一个环形反射光栅. (如下图为光盘对红色激光的衍射, 可以清楚地看见不同级次的衍射光线)


    (右图来自http://laserpointerforums.com/f48/diffraction-grating-33864.html )

    但是衍射角还和波长有关, 对于不同的波长同一级次的衍射角并不相同. 所以当非单色波入射到光栅上时会出现色散. CD 看起来是彩色的正是光栅的色散效应. 下图为一个透射光栅的色散


    (图片来自http://braindanceisawayoflife.blogspot.com/2012/04/higher-frequency-photon-has-greater.html )
