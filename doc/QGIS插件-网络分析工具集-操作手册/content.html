01. 插件简介  
QGIS 网络分析工具集是一款 QGIS 插件，为本小组 4 人所设计和开发，目  
前是第一版，该插件的目的是将独立的地理网络的分析工具集成起来，第一版集  
成了五项工具，我们对第二版的最大期望是能够让用户方便地把自定义的工具集  
成进来。  
 优点及特色  
(1) 安装包很轻巧，安装简单；  
(2) 易于管理，有 QGIS 的 plugin manager 和 plugin reloader 的支持；  
(3) UI 简洁美观，基本上一看就懂；  
(4) 除了“最短路径分析”、“最小生成树计算”这些常见的网络分析工具，  
还提供有高级网络分析工具，如“线/面最小跨度生成三角网”、“重要节  
点提取”以及“根据节点位置划分网络”；  
(5) 每个工具的参数设置较少、容易理解，而且给出了中文的解释说明；  
(6) 插件运行稳定、高效；  
(7) 实时的运行日志输出帮助用户进行错误监控；  
(8) 具有一定的可扩展性。源码中功能按钮的处理槽都很容易定位，而且网  
络分析的核心算法是独立于处理槽的，用户只要理解了处理槽函数中少  
量的代码就能将自己的网络分析算法集成到插件中；  
 环境要求  
本插件在 QGIS 3.6 中开发，运行本插件的最低 QGIS 版本为 QGIS 3.0  
（详见插件的元数据文件：metadata.txt）  
 github 地址  
主页：https://github.com/Team-For-WebGIS/Network_Toolkit  
Clone: https://github.com/Team-For-WebGIS/Network_Toolkit.git  
Zip 下载：https://github.com/Team-For-WebGIS/Network_Toolkit/archive/master.zip  
02. 插件管理  
 安装  
下载安装包后解压至 QGIS 插件安装目录，可以是 QGIS 默认的插件安  
1
装目录也可以是用户在环境变量中自己配置的插件安装目录。在 win10 下  
QGIS 的默认插件安装目录为：  
C:\Users\username\AppData\Roaming\QGIS\QGIS3\profiles\default\python\  
plugins\  
解压后重启 QGIS 使插件安装生效。然后就能在 QGIS 的 plugin manager  
中找到安装后的插件 “ Network Toolkit“。  
 卸载  
和 QGIS 其它插件的卸载一样，在 plugin manager 中找到该插件，选中后  
点击右下角的卸载按钮。  
 重载  
推荐使用 QGIS 的 plugin reloader 进行插件的重载，除此之外，在 plugin  
manager 中重新勾选该插件或者重启 QGIS 也能达到重载插件的目的。

05. 常见错误与解决方法  
 错误一：插件解压到目录后 plugin manager 找不到  
QGIS  
—— 如果确认解压目录在  
的插件文件夹下，请查看确保文件被解  
压到的目录下就是插件的源码和资源，解压软件可能会多加一层文件夹。  
 错误二：工具运行出错，日志显示输出文件错误  
—— 请确认在参数中输入的图层不是空的，输入的图层中至少要有一个  
要素。  
 错误三：最短路径运行结束后日志显示找不到最短路径  
—— 如果在用户看来选择的起点和终点确实是相连的，请确保选点时准确  
点击在道路上，离道路太远的话，算法无法将选择的点加入到网络中。  
 错误四：最小跨度三角网和最小生成树的生成结果明显有误。  
—— 由于算法依赖名称为 point 和 line 的图层，在运行该工具前，请为工程  
中名称为 point 和 line 的图层改名，确保在工具运行时没有这两个图层。
